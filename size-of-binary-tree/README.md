### Size of Binary Tree

Given the root of a binary tree, return the size of the tree. The size of a binary tree is the total number of nodes in the tree.

Examples:

![input1](https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/926891/Web/Other/blobid0_1776851879.png)

```
Output: 3
Explanation: There are 3 nodes in the given binary tree, so its size is 3.
```

![input2](https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/926891/Web/Other/blobid1_1776851914.png)

```
Output: 6
Explanation: There are 6 nodes in the given binary tree, so its size is 6.
```

Constraints:

```
1 ≤ number of nodes ≤ 10^5
1 ≤ node->data ≤ 10^5
```
